# md5-lite
MD5 hashing with part of a file

## Installation 
```sh
npm install md5-lite --save
yarn add md5-lite
bower install md5-lite --save
```
## Usage

### Javascript
```javascript
const md5 = require('md5-lite');
console.log(md5('./filepath'));
```
```sh
0486c21a4d40d0f3b0bdad3925db439e
```

### TypeScript
```typescript
import md5 from 'md5-lite';
console.log(md5('./filepath'))
```
```sh
0486c21a4d40d0f3b0bdad3925db439e
```
